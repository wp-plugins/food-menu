Create and manage food menus in Wordpress.

== Description ==

This plugin is designed for cafes, pubs, hotels and restaurants to add their own food menus to their Wordpress site.

The plugin allows you to create your own courses and the add dishes to those courses. Each dish has a title, description and price and can be tagged. The dishes are then added to one or more of the courses.

The menu feature in Wordpress can then be used to create pages for each course. Courses can be added to pages and posts using shortcode if needed.

The plugin was designed to work with the Genesis Theme Framework. 


== Installation ==

1. Upload the 'food-menu-plugin' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you use Genesis framework
